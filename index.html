Installation

1
Dans Guerre Tribale, va dans Paramètres → Barre de raccourcis et clique sur Nouveau raccourci.
2
Clique sur le bouton de copie du script voulu ci-dessus.
3
Colle le code dans le champ URL du raccourci. Donne-lui un nom et sauvegarde.
4
Navigate vers l'écran requis (indiqué sur chaque carte), puis clique sur le raccourci pour exécuter le script.
